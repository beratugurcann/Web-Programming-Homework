Proje Özeti

Bu proje, bu "https://www.youtube.com/watch?v=HGTJBPNC-Gw&t=10285s" videodaki uygulamaları yapmak için hazırlanmıştır. HTML ve CSS kullanılarak sayfa tasarımı, butonlar, linkler ve multimedya içeriklerinin nasıl çalıştığı gösterilmiştir.

/WebProg
│
└─ Ödev/
   ├─ main.html       # Ana sayfa
   └─ html/           # Diğer HTML dosyaları
       ├─ css/        # Stil dosyaları
       └─ assets/     # Resim, video ve diğer medya dosyaları

Çalıştırma

Ödev/main.html dosyasını bir web tarayıcısında açın (Chrome, Firefox, Edge vb.).

Dosya yolları doğruysa, sayfa sorunsuz şekilde çalışacaktır.
